Как добавить новый файл?
"Очень просто." - хотелось бы так сказать, пока не могу. Но это должно быть простым для всех нас.

Исходное состояние:
  - у Вас есть аккаунт на ГитХаб;
  - у Вас есть желание участвовать в проекте;
  - у Вас есть информация, которую Вы хотите добавить к проекту;
  - у Вас есть ссылка на проект: https://github.com/Club-BV225 ;
  - Вы находитесь в своем аккаунте ГитХаба.
  
 Теперь собственно операция по добавлению нового файла в общий проект:
  - переходите по ссылке на страницу общего проекта;
  - выбираете папку, в которую хотите добавить новый файл (выбираете проект);
  - справа на дополнительной панели (не в верхнем правом углу, а заметно ниже) находите "fork" и нажимаете. Откроется дерево структуры всех ветвей (сейчас их немного);
  - выбираете свою ветку (содержащую в своем названии имя Вашего аккаунта). Откроется страница проекта, такая же, как в общем проекте;
  - в верхней части нажимаете "add ytw file...". Откроется страница редактирования файла;
  - даете имя своему файлу в строке "Project_1/Имя_Вашего_файла.txt in main";
  - в поле "Edit new file" пишете или вставляете скопированный текст;
  - снизу в поле "Commit new file" добавляете описание коммита;
  - нажимаете кнопку ниже "Commit new file";
Вы создали новый файл. Молодцы! Правда. Теперь нужно создать запрос на вытягивание (create pull reqгest). Типа запрос на включение файла в общий проект.
После этого все могут обсудить этот файл (не знаю как эта процедура происходит, во всяком случае, можно ознакомиться с файлом в общем проекте).
НО! файл ещё не включен в общий проект и на главной странице его не видно. Доступ к файлу можно получить через вкладку "Pull request". Там можно ознакомиться с внесенными изменениями. Это вроде сборника черновиков, не утвержденных быть в главной ветке (пока не утвержденных).
Дальше участник, имеющий полномочия на слияние веток (пока что это только я) выполнит "утверждение" вашего файла и включит его в общий проект. То есть, выполнит слияние веток.
